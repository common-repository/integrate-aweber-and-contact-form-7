== Description ==

**Integrate AWeber and Contact Form 7**

Features:

*   Easy to use
*   Send and save subscriber data to AWeber account
*   Supports multiple accounts and lists
*   Simple to map contact form fields(mail tags) to AWeber list fields
*   Fast and secure
*   Uses latest aweber api with OAuth 2.0
*   Compatible with Contact Form 7 (4.2 or higher)

== Installation ==

1. Upload the entire `cf7-awb` directory to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use `Aweber` tab in your Contact Form 7 editor to configure the plugin.
